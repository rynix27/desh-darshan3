# DeshDarshan — Incredible India Travel Website
## Tech Stack
- Frontend : HTML5, CSS3, JavaScript
- Backend  : Java 17 + Spring Boot 3.2
- Database : MySQL 8.0

## Frontend Files (open index.html to start)
- index.html           Home
- destinations.html    22 Destinations with map
- accommodation.html   Hotels with filters and map
- culture.html         Festivals, Dances, Shows, Cuisine, Crafts
- trip-planner.html    Trip cost calculator and itinerary
- travel-tips.html     Travel tips and external links
